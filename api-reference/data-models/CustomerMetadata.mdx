## `CustomerMetadata` Class

| Property        | Type           | Optional | Description                                    |
|-----------------|----------------|----------|------------------------------------------------|
| `customBranding`| CustomBranding | Yes      | Configuration for custom branding elements.    |

## `CustomBranding` Class

For the `CustomBranding` class, due to its nested structure, the table below describes the top-level `logo` property, and subsequent tables break down the nested objects:

| Property | Type  | Optional | Description                    |
|----------|-------|----------|--------------------------------|
| `logo`   | Object| Yes      | Configuration for custom logos.|

### Logo Object in `CustomBranding` Class

| Property   | Type   | Optional | Description                      |
|------------|--------|----------|----------------------------------|
| `logoMark` | Object | Yes      | Configuration for the logo mark. |
| `wordMark` | Object | Yes      | Configuration for the word mark. |

### `LogoMark` and `WordMark` Objects Class

Each of these objects (`logoMark` and `wordMark`) have the same structure:

| Property  | Type   | Optional | Description                                 |
|-----------|--------|----------|---------------------------------------------|
| `lightUrl`| string | Yes      | URL of the light mode version of the logo.  |
| `darkUrl` | string | Yes      | URL of the dark mode version of the logo.   |
