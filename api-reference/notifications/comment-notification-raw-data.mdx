---
title: "Comment Notification Raw Data"
---

| Field               | Type              | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| ------------------- | ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| ' commentAnnotation | CommentAnnotation | The target CommentAnnotation object on which the event happened.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| targetComment       | Comment           | The target Comment object on which the event happened. This field will not be present if the event was at CommentAnnotation level. Eg: deleted the entire comment annotation, resolved comment, approved comment.                                                                                                                                                                                                                                                                                                                                                                                |
| actionType          | string            | This can have 5 values: added, updated, deleted, resolved, approved - added: when a new Comment is added. - updated: when an existing Comment is updated. - deleted: when an existing Comment or the entire CommentAnnotation is deleted. If CommentAnnotation is deleted then commentText field will be empty. - resolved: when theCommentAnnotation is resolved. commentText field will be empty in this case. - approved: when theCommentAnnotation is approved by the moderator. This is only applicable if you have turned on Moderator Mode. commentText field will be empty in this case. |
