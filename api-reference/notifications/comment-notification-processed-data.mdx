---
title: "Comment Notification Processed Data"
---

| Field             | Type          | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| ----------------- | ------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| commentText       | string        | Comment text for the added, updated or deleted Comment. This is empty when entire CommentAnnotation is deleted.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| commentTextFrom       | UserData       |  user of updated comment of last comment   |
| toContactsTagged   | UserContact[]  | tagged user contacts |
| toContactsUpdated   | UserContact[]  | user contacts to be updated |
| from              | User          | The user who added, updated or deleted the target Comment.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| toContactsTagged  | UserContact[] | List of user contacts that are tagged to the target Comment.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| toContactsUpdated | UserContact[] | List of user contacts that are on the target CommentAnnotation. This excludes all the user contacts that are on the target Comment.                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| pageTitle         | string        | Title of the page on which the target CommentAnnotation is present.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| pageUrl           | string        | Full URL of the page on which the target CommentAnnotation is present.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| actionType        | string        | This can have 5 values: added, updated, deleted, resolved, approved - added: when a new Comment is added. - updated: when an existing Comment is updated. - deleted: when an existing Comment or the entire CommentAnnotation is deleted. If CommentAnnotation is deleted then commentText field will be empty. - resolved: when the CommentAnnotation is resolved. commentText field will be empty in this case. - approved: when the CommentAnnotation is approved by the moderator. This is only applicable if you have turned on Moderator Mode. commentText field will be empty in this case. |
| status            | string        | comment annotation status
| notificationSource? | string      | comment (source of CommentNotificationRawData)
| slackChannelId?   | string        | slack channel id (optional)
| newPriority?      | string or any     | new priority (if priority is changed)
| documentId? | string      | velt document id
| clientDocumentId? | string      | client document id
| deviceInfo? | any   |  device info we get from document metadata from RawNotificationData
