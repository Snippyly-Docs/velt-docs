---
title: "Selection Annotation"
---

| Field           | Type                                           | Description                                    |
| --------------- | ---------------------------------------------- | ---------------------------------------------- |
| user            | User \| undefined                              | User focused on specific element or text       |
| type            | 'focus' \| 'click' \| 'selection' \| undefined | Type of event through highlight event is fired |
| targetElement   | string \| null \| undefined                    | XPath of highlighted element                   |
| targetTextRange | TargetTextRange \| null \| undefined           | Selected text to highlight                     |
| pageInfo        | PageInfo                                       | Page Information                               |
