title: "Comment Element"

The CommentElement class contains the following fields:

### 1. getAllCommentAnnotations()

This method is used to subscribe to a list of all comment annotations on the current document or a specified document.

**Signature**

```java
public getAllCommentAnnotations(documentId?: string, location?: Location): Observable<CommentAnnotation[] | null>;
```

**Params**

`documentId: string | undefined - The ID of the document to get comment annotations from. If not specified, the current document is used.`

`location: Location | undefined - The location within the document to get comment annotations from. If not specified, the entire document is searched.`

**Returns**

`Observable<CommentAnnotation[] | null>`
