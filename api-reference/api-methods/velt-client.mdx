## initConfig

Initialize Snippyly configuration with the provided API key and optional configuration.

```javascript
client.initConfig(apiKey: string, config?: Config)
```

## identify

Identify the user and return a Promise with an unknown result. Useful for tracking user activity.

```javascript
client.identify(user: User, userOptions?: UserOptions)
```

## updateUser

Update information about the currently logged-in user.

```javascript
client.updateUser(user: User)
```

## setDocumentId

Set the unique document ID for the current document/resource to enable collaboration.

```javascript
client.setDocumentId(id: string)
```

## setLocation

Set custom parameters for the current document/resource, allowing sub-document identification.

```javascript
client.setLocation(params: Location, appendLocation?: boolean)
```

## removeLocation

Remove a location from the user, which may include sub-document identification.

```javascript
client.removeLocation(location?: Location)
```

## setScrollableElementsIds

Inform Snippyly about scrollable elements' IDs to track their scroll changes.

```javascript
client.setScrollableElementsIds(ids: string[])
```

## removeScrollableElementsIds

Remove scrollable elements' IDs from tracking.

```javascript
client.removeScrollableElementsIds()
```


## getTagElement

Retrieve the Tag Pin Annotation Object for accessing raw tag data, suitable for advanced users who want to design their own UI.

```javascript
client.getTagElement()
```

```javascript
client.getSelectionElement()
```

## signOutUser

Sign out the current user.

```javascript

```

## disableFeatures

Provide a list of features to disable or an empty array to enable all features.

```javascript
client.disableFeatures(features: Array<FeatureType>)
```

## addLocation

Add a location to show comments, tags, recorders, etc., for the provided location. Deprecated, use `setLocation` with `appendLocation: true` instead.

```javascript
client.addLocation(location: any)
```

## isUserAllowed$

Check if the user is allowed in the provided document. Returns an Observable.

```javascript
client.isUserAllowed$()
```

## getUserRole$

Get the user's role in the provided document. Returns an Observable.

```javascript
client.getUserRole$()
```

## isPlanExpired$

Check if the plan is expired or not. Returns an Observable.

```javascript
client.isPlanExpired$()
```

## inviteUsers

Invite a list of users to the document.

```javascript

```

## removeSnippylyContent

## removeVeltContent


## disableAutoTranslation
