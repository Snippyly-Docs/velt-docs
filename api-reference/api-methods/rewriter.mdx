


<Steps>
<Step title="enableRewriter()">
This method enables the rewriter.

Example:
```jsx
const rewriterElement = client.getRewriterElement();
rewriterElement.enableRewriter();
```

**Signature:** 
```
enableRewriter() => void
```

**Params:** 
```
none
```

**Returns:** 
```
void
```


</Step>

<Step title="disableRewriter()">

Example:
```jsx
const rewriterElement = client.getRewriterElement();
rewriterElement.disableRewriter();
```


**Signature:** 
```
disableRewriter() => void
```

**Params:** 
```
none
```

**Returns:** 
```
void
```

</Step>
