To enable rewriter feature.

**Signature**

```java

```

**Params**

`None`

**Returns**

`void`

2. disableRewriter()
   To disable rewriter feature.

**Signature**

```java
public disableRewriter: () => void;
```

**Params**

`None`

**Returns**

`void`
