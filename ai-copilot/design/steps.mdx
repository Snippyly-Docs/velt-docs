### 1) Enable Rewriter 

First, import `useVeltClient` from `@veltdev/react`.

Then create a client object using the `useVeltClient` hook.

Lastly, within a `useEffect` hook, call getRewriterElement() and then enableRewriter() to enable Rewriter.

```react
import { useEffect } from "react";
import { useVeltClient } from "@veltdev/react";

export default function App() {
  let { client } = useVeltClient();

  useEffect(() => {
    if (client) {
      const rewriterElement = client.getRewriterElement();
      rewriterElement.enableRewriter();
    }
  }, [client]);

  return (
    // your app template
  );
}

```
