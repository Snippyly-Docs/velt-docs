---
title: "Rewriter Annotation"
---

| Field           | Type                          | Description                                                  |
| --------------- | ----------------------------- | ------------------------------------------------------------ |
| annotationId    | string                        | Unique identifier for the rewriter pin annotation            |
| from            | User                          | The user who created this rewriter pin annotation            |
| color           | string \| undefined           | Color used for the rewriter pin annotation                   |
| lastUpdated     | any \| undefined              | Timestamp when the rewriter annotation was last updated      |
| location        | Location \| null \| undefined | Set location to identify user on sub document                |
| type            | string                        | Type of the annotation                                       |
| rewriterType    | 'generic' \| 'copywriter'     | Type of the rewriter                                         |
| annotationIndex | number \| undefined           | Index of current annotation in available list of annotations |
| pageInfo        | PageInfo                      | Page Information                                             |
