---
title: "Area Annotation"
---

| Field           | Type                          | Description                                         |
|-----------------|-------------------------------|-----------------------------------------------------|
| annotationId    | string                        | Unique identifier for the area pin annotation       |
| from            | User                          | The user who created this area pin annotation       |
| color           | string \| undefined           | Color used for the area pin annotation              |
| lastUpdated     | any \| undefined              | Timestamp when the area annotation was last updated |
| location        | Location \| null \| undefined | Set location to identify user on sub document       |
| type            | string                        | Type of the annotation                              |
| annotationIndex | number \| undefined           | Index of current annotation                         |
| pageInfo        | PageInfo                      | Page Information                                    |
