## 1) Install React package

npm:
```bash
$ npm install @veltdev/react
```

yarn:
```bash
$ yarn add @veltdev/react
```


## 2) Install types (optional)

If you're using TypeScript, you can install the types package.

```bash
$ npm install --save-dev @velt/types
```

## 3) Add VeltProvider

Add the VeltProvider component to the root of your app.
Add your Velt API key.


```js
<VeltProvider apiKey="YOUR_API_KEY">
      // Your app goes here
</VeltProvider>
```

  <Tab title="HTML">

   ### 1) Import the Velt Script tag
    ```html
    <script type="module" src="https://cdn.jsdelivr.net/npm/@veltdev/sdk@1.0.11/velt.js" onload="loadVelt()"></script>
    ```

    ### 2) Initialize with your API Key

    Put this in your root app script:
    ```js
    async function loadVelt() {
        await Velt.init("YOUR_VELT_API_KEY");
    }
    ```
