description: "Document is the space where you want your users to collaborate.For a design tool it's the canvas, for a video editor, it's the editor, for a documentation tool, it's the document etc. A document is like a virtual house. Users in the same virtual house can see and interact with each other."

## 1) Get the Velt client

Import the `useVeltClient` React hook.
You can use this hook within your component to fetch the Velt client.

```js
import { useVeltClient } from '@veltdev/react';
```

## 2) Create a useEffect hook

Create an effect with the client as a dependency.
Make sure to check if the client is `null` or `undefined` before you use it.

```js
useEffect(() => {
  if (client) {
    //set document ID
  }
}, [client]);
```

## 3) Set a document ID

Pass a unique ID representing your document. Whenever your document initializes your canvas is ready for collaboration, you should call this method. The SDK will not work without this call.
Users logged into the same document ID can see each other's presence, cursors, comments etc.


```js
  client.setDocumentId('unique-document-id');
```

import { useVeltClient } from '@veltdev/react';
import { useEffect, useState } from 'react';

export default function YourCollaborationApp() {

  const { client } = useVeltClient();

  useEffect(() => {
    if (client) {
      client.setDocumentId('unique-document-id');
    }
  }, [client]);

  return (
    // Your app template
  );
}
