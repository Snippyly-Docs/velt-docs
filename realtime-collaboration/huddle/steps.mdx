  <Tab title="HTML">
### 1) Import the Velt Script tag
```html
<script type="module" src="https://cdn.jsdelivr.net/npm/@veltdev/sdk@1.0.11/velt.js" onload="loadVelt()"></script>
```

### 2) Initialize with your API Key

Put this in your root app script:
```js
async function loadVelt() {
    await Velt.init("YOUR_VELT_API_KEY");
}
```

### 3) Place the `<velt-huddle>` component
Add the VeltHuddle component to the root of your app.

This component is required to render huddle UI components & huddle users in your app.

### 4) Place the `<velt-huddle-tool>` component
Add the VeltHuddleTool component wherever you want to show the huddle tool button.

Clicking on it initiates a huddle.
