## Live State Sync

With Live State Sync, you can create a data object that persists and syncs across different clients.

## Syncing Text elements

To allow Input, Textarea or Contentedible elements to sync across clients, set the `data-velt-sync-state` attribute to `true` on the elemement you are trying to sync:


```jsx
<textarea data-velt-sync-access="true"></textarea>
```


## Disabling Clicks on elements

To disable clicks on elements, set the `data-velt-sync-access` attribute to `false` on the elemement you are trying to disable:

```jsx
<div data-velt-sync-access="false"></div>
```
