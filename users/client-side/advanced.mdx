## Reset Contact List

To reset a user's contact list, use the `client.identify()` mehtod and pass in `{replaceContacts: true}` as the second parameter.

```jsx
client.identify(yourLoggedInUser, {
    replaceContacts: true
}).then((res) => {
  // User auth successful with Velt
}).catch((err) => {
  // Failed to authenticate user with Velt
});
```
