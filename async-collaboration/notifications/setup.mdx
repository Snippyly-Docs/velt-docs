<Tabs>
  <Tab title="React / Next.js">

<Steps titleSize="h2">

<Step title="Enable Notifications in the Velt Console">

Go to the [Notifications section](https://console.velt.dev/dashboard/config/notification) in the Configurations section of the Velt Console and enable Notifications.

Notifications will not work if you do not do enable them first.


</Step>


<Step title="Import the VeltNotificationsTool and VeltNotificationsHistoryPanel component from @veltdev/react">

Import the `VeltNotificationsTool` and `VeltNotificationsHistoryPanel` components.

```js
import { VeltNotificationsTool, VeltNotificationsHistoryPanel } from '@veltdev/react';
```
</Step>


<Step title="Place the VeltNotificationsTool and VeltNotificationsHistoryPanel components">
Place the `VeltNotificationsTool` component wherever you want the `Notifications` button to appear.

You can place the `VeltNotificationsHistoryPanel` anywhere, as it will appear from the side when enabled.

```jsx
<div className="toolbar">
  <VeltNotificationsTool/>
  <VeltNotificationsHistoryPanel/>
</div>
```
</Step>



</Steps>


  </Tab>

  <Tab title="HTML">



<Steps titleSize="h2">

<Step title="Enable Notifications in the Velt Console">

Go to the [Notifications section](https://console.velt.dev/dashboard/config/notification) in the Configurations section of the Velt Console and enable Notifications.

Notifications will not work if you do not do enable them first.


</Step>

<Step title="Import the Velt Script tag">
```html
<script type="module" src="https://cdn.jsdelivr.net/npm/@veltdev/sdk@latest/velt.js" onload="loadVelt()"></script>
```
</Step>

<Step title="Initialize with your API Key">
Put this in your root app script:
```js
async function loadVelt() {
    await Velt.init("YOUR_VELT_API_KEY");
}
```
</Step>


<Step title="Place the velt-notifications-tool and velt-notifications-history-panel components">
Place the `<velt-notifications-tool>` component wherever you want the `Notifications` button to appear.

You can place the `<velt-notifications-history-panel>` anywhere, as it will appear from the side when enabled.


```jsx
<div className="toolbar">
  <velt-notifications-tool></velt-notifications-tool>
  <velt-notifications-history-panel></velt-notifications-history-panel>
</div>
```
</Step>



</Steps>



  

  </Tab>
</Tabs>
