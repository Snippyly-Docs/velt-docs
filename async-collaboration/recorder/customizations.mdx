### Customizations

## Custom CSS
You can override the CSS of the entire component yourself.

## Custom Button
You can create your own button instead of using the button we provided. To do so, pass in your button as a child component to `VeltRecorderTool`

```js
import { VeltRecorderTool } from '@veltdev/react'

const SomeComponent = () => {
  return (
    //... rest of the component code...

		<VeltRecorderTool type="all">
            <button>Start Recording</button> //you can add any html code here.
        </VeltRecorderTool>
    
    //... rest of the component code...
  )
}

export default SomeComponent;
```

## Button Labels

To add a custom label to the VeltRecorderTool pass a string to the buttonLabel attribute.

```js
import { VeltRecorderTool } from '@veltdev/react'

const SomeComponent = () => {
  return (
  	//... rest of the component code...
    <VeltRecorderTool type="all" buttonLabel="Add Recording" />

    //... rest of the component code...
  )
}

export default SomeComponent;
```

## onDelete callback

The creator of the recordings sees the option to delete the recording. Once the user deletes the recording, we publish a callback method called onDelete with the recorder ID. You can listen to the callback and use the response object to perform any clean up on your end. This is the response object that you get back:

| Field    | Type                          | Example       |
| -------- | ------------------------------| --------------|
| id       | string                        | "recorder_id" |

```js
import { VeltRecorderPlayer } from '@veltdev/react'

const SomeComponent = () => {
  
  const onDelete = (data) => {
    console.log('on delete: ', data);
  }
   
  return (
  	//... rest of the component code...
		<VeltRecorderPlayer recorderId={RECORDER_ID} onDelete={(data) => onDelete(data)} />

    //... rest of the component code...
  )
}

export default SomeComponent;
```
