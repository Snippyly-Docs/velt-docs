## Custom Arrow 

If you want to replace the default arrow with your own custom component, you can pass it in as a child component.

<Tabs>
    <Tab title="React">

```js React
import { VeltArrowsTool } from '@veltdev/react';

function YourComponent() {

return (
        ...
    <VeltArrowsTool>
        //custom component goes here
    </VeltArrowsTool>
    ...
  )
  
}
```


    </Tabs>
    <Tab title="HTML">
<velt-arrows-tool>
//custom arrow html goes here
</velt-arrows-tool>
    </Tabs>
</Tabs>
