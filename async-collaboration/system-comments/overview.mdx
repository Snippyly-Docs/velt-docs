## 1) Visit the portal

Open your web browser and navigate to snippyly-sdk-portal.web.app.

## 2) Access the configurations menu

Once you've logged in, locate and click on the [Configuration](https://snippyly-sdk-portal.web.app/dashboard/config) menu on the homepage or dashboard. This should be somewhere at the top or side of the page, depending on the website's layout.

## 3) Generate an authentication token

Inside the [Configuration](https://snippyly-sdk-portal.web.app/dashboard/config) menu, find the 'Auth token' option.

If a token hasn't been generated yet, click on the 'Generate token' option.

## 4) Copy the generated token

After clicking on 'Generate token', a token should be displayed on the screen.

Copy this token for later use. Make sure not to share it with anyone as it can provide access to your personal data.

## 5) Prepare the POST API request

Now, prepare a POST API request using the JSON data mentioned as a reference in the side panel.

Replace "your_token_here" with the token you copied earlier. Replace 'apiKey', 'documentId' and other values with your actual data.

## 6)  Send the POST API request

Finally, send the POST API request to `https://us-central1-snipply-sdk-staging.cloudfunctions.net/addSystemComment` API endpoint.

If the request is successful, you should receive a 200 OK HTTP status code, along with a response from the server confirming that the comment has been added.

<ResponseField name="apiKey" type="string">
The API key for the project
</ResponseField> 

<ResponseField name="authToken" type="string">
The authentication token generated from the configurations menu
</ResponseField>

<ResponseField name="documentId" type="string">
The document ID where the comment was added
</ResponseField>

<ResponseField name="targetElement" type="object">
The target element on which you want to add comment
</ResponseField>

<ResponseField name="commentData" type="object[]">
The comment data to be added
</ResponseField>

<ResponseField name="location" type="string">
The location where the comment was added
</ResponseField>
