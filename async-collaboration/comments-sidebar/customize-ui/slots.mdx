## 1) Replace the Sidebar Button

**Provide a template for the sidebar button.**

Target the button slot with your own custom template.

<RequestExample>

```java React
import {
  SnippylySidebarButton
} from '@veltdev/react';

export default function App() {

  return (
    <>

      <SnippylySidebarButton>
        <div class="sidebar-button">
          <img src="..." />
        </div>
      </SnippylySidebarButton>

    </>
  );
}
```
