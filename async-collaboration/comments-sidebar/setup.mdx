<Tab title="React">
## 1) Import VeltCommentsSidebar and VeltSidebarButton ffom `@veltdev/react`
**Import the comments sidebar component.**

```js
import { 
  VeltProvider, 
  VeltCommentsSidebar, 
  VeltSidebarButton,
  VeltCommentTool 
} from '@veltdev/react';
```

We offer specific components for our users that use React.

## 2) Add Comments and Sidebar components
**Add the VeltComments and VeltCommentsSidebar components to the root of your app.**

```js
<VeltComments />
<VeltCommentsSidebar />
```

This component is required to render comments in your app.

## 3) Add Sidebar button
**Add the Sidebar button to toggle the sidebar.**

```js
<VeltSidebarButton />
```

This is completely optional and you can toggle the sidebar in the comment dialog as well.

## 4) Test Integration
**Test it out by opening the sidebar.**

You should be able to click the `All comments` link in a comment dialog box on the bottom.

<Tab title="HTML">
    ### 1) Import the Velt Script tag
    ```html
    <script type="module" src="https://cdn.jsdelivr.net/npm/@veltdev/sdk@1.0.11/velt.js" onload="loadVelt()"></script>
    ```

    ### 2) Initialize with your API Key

    Put this in your root app script:
    ```js
    async function loadVelt() {
        await Velt.init("YOUR_VELT_API_KEY");
    }
    ```

    ### 3) Place the `<velt-comments-sidebar>` component
    Place the component wherever you want the invite button to appear.

```html HTML
<velt-comments-sidebar></velt-comments-sidebar>
```

    ### 4) Place the `<velt-sidebar-button>` component
    Place the component wherever you want the invite button to appear.

```html HTML
<velt-sidebar-button></velt-sidebar-button>
```
