<Tabs>
<Tab title="Pin">
## 1) Select the part you want to modify

We offer several parts which can be used like classes. Full list below.
The component is encapsulated in Shadow DOM, which is isolated from the normal DOM.

## 2) Apply your CSS rules

Set whatever CSS rules you want.
The part lets you target a specific element within a Shadow DOM.

## 3) Check out the table below

Reference the table below to see what parts we expose.
Alternatively, you can directly inspect the component HTML to see what parts are available.

<ResponseField name="comment-pin-bubble">
Targets bubble comment pin container
</ResponseField>

<ResponseField name="comment-pin-popover"> 
Targets the popover comment pin container
</ResponseField>

<ResponseField name="comment-pin">
Targets the default comment pin container
</ResponseField>

<ResponseField name="index-container">
Targets the comment pin index label container
</ResponseField>

<ResponseField name="index-container-span">
 Targets comment pin index label
</ResponseField>

<ResponseField name="lock-container">
Targets the comment pin internal lock container
</ResponseField>

<ResponseField name="lock-icon">
 Targets the comment pin internal lock icon
</ResponseField>
</Tab>

<RequestExample>
```css Pin
snippyly-comment-pin::part(lock-icon) {
  width: 0.8rem;
  height: 0.8rem;
  color: grey;
}
```
```css Bubble
snippyly-comment-bubble::part(count) {
  font-size: 14px;
}
```
