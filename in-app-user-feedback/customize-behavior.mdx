### Feedback Mode
By default the component will be in Feedback mode. This will make your component say "Give Feedback" and have a heart icon.

### Report Bugs Mode
To enable Report Bugs mode, set the type attribute to 'reportBug`. This will make your component say "Report Bugs" and have a bug icon.

<iframe
      src="https://snippyly-docs-demo.web.app/report-bug?documentId=user-request-docs&amp;userIndex=0"
      scrolling="no"
      frameborder="0"
      class="w-full"
></iframe>
</Frame>


### Contact Us Mode
To enable Report Bugs mode, set the type attribute to 'contactUs`. This will make your component say "Contact Us" and have an email icon.
